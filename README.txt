FAST MS (Free Analysis Software for Top-down Mass Spectrometry) is a software package for data analysis of MS and MS/MS spectra.
You may consider the publication (https://pubs.acs.org/doi/10.1021/jasms.4c00236), the manual ("FAST MS Manual.pdf") or my dissertation (https://ulb-dok.uibk.ac.at/ulbtirolhs/content/titleinfo/8051320; chapter 3 "FAST MS: An Open-Source Software for Automated and Quantitative Analysis of Mass Spectra" in "RNA Structure Characterisation by "Chemical Probing and Mass Spectrometry"). Please, cite the paper when you use the programme.
If you have a Windows operating system, you can use the executable version which can be found in Releases (https://github.com/michael-palasser/FAST-MS/releases). The current version is 1.0.0.
